# Stage Gate Review Checklist

每个阶段进入下一阶段前都必须完成本清单。

## Stage 信息

- Stage：
- 对应 task：
- 审查人：
- 审查日期：
- 结论：pass / revise / fail

## 通用门槛

- [ ] 本阶段目标已经完成；
- [ ] 没有提前实现后续阶段；
- [ ] README / ROADMAP / docs 与实现一致；
- [ ] 自动检查通过或失败原因已解释；
- [ ] 所有新增实验或场景可复现；
- [ ] 没有提交大文件或未筛选产物。

## 研究有效性

- [ ] 变更仍服务于 latent motion intent uncertainty；
- [ ] 没有退化成普通避障或普通动态避障；
- [ ] 新增指标与 `docs/04_metrics_contract.md` 一致；
- [ ] 新增场景与 `docs/05_scenario_benchmark.md` 一致；
- [ ] 结论没有超出实际实验支持。

## 下一阶段准备

- [ ] 下一阶段 task 已明确；
- [ ] 阻塞问题已记录；
- [ ] 已知限制已记录；
- [ ] 必要的审查产物已链接或说明。
