# 场景可视化审查清单

每个候选测试 seed 必须审查以下项目。

## 基本合理性

- [ ] 起点和终点没有生成在障碍物内部。
- [ ] 起点到终点存在至少一条可行路径。
- [ ] 场景不是直线无风险，也不是必撞无解。
- [ ] easy / medium / hard 难度差异明显。

## 风险有效性

- [ ] 风险对象未来轨迹与无人机名义走廊有潜在交集。
- [ ] sudden_start 对象不是全部启动，也不是全部保持静止。
- [ ] occluded_emergence 中遮挡确实成立。
- [ ] intent_ambiguous 中早期轨迹保持模糊。
- [ ] semantic_distractor 中对象不应真正进入主路径。

## 可视化完整性

- [ ] layout 图清楚显示起点、目标、障碍物和对象。
- [ ] timeline 图清楚显示对象启动或出现时刻。
- [ ] 多 seed 网格图能帮助快速发现异常。
- [ ] 图例统一。

## 结论

- [ ] PASS
- [ ] FIX
- [ ] REJECT
