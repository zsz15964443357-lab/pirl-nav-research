# TASK 02：可视化流水线

## 目标

为每个候选场景 seed 自动生成可审查图像。

## 必须先读

- `docs/05_可视化审查规范.md`
- `experiments/review_checklists/场景可视化审查.md`

## 必须输出

每个候选场景至少生成：

1. 俯视布局图；
2. 对象轨迹时间线；
3. 多 seed 网格预览；
4. Markdown 或 HTML 索引页。

## 图例规范

- UAV 起点：蓝色圆点；
- 目标：绿色星标；
- 静态障碍物：灰色块；
- 遮挡物：黑色块；
- 动态对象：橙色点；
- 潜在风险对象：红色点；
- 名义路径走廊：蓝色虚线；
- 对象未来轨迹：箭头。

## 禁止事项

- 不开始训练；
- 不把未审查 seed 加入固定测试集。

## 审查门槛

每类场景每个难度至少抽查 10 个 seed。
