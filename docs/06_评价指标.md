# 06 评价指标

## 主表指标

论文主表至少报告：

| 指标 | 含义 | 越大越好 |
|---|---|---|
| success rate | 成功到达目标比例 | 是 |
| collision rate | 碰撞比例 | 否 |
| near-miss rate | 近失比例 | 否 |
| integrated risk exposure | 穿越未来高风险区域的累计暴露 | 否 |
| Anticipation Time | 策略相对 TTC 危险阈值提前响应时间 | 是 |
| Shield Dependence Index | 策略对 safety shield 的依赖程度 | 否 |
| jerk | 动作抖动或加速度变化 | 否 |
| detour ratio | 绕行比例 | 否 |

## Anticipation Time

用于衡量策略是否真正提前避险。

```text
AT = t(TTC < TTC_threshold) - t(policy_risk_response > response_threshold)
```

AT 越大，说明策略越早开始响应潜在风险。

## Shield Dependence Index

用于衡量策略是否依赖末端安全修正。

```text
SDI = mean_t ||a_safe,t - a_policy,t|| / (||a_policy,t|| + epsilon)
```

SDI 越低，说明策略越少依赖 safety supervisor。

## 风险暴露

risk exposure 衡量无人机候选轨迹或实际轨迹与未来概率占据区域的重叠。

这个指标必须与 action-conditioned risk 保持一致，不应只是语义类别权重。

## 分场景报告

不要只报告整体平均。必须按以下场景分别报告：

- static_clutter；
- dynamic_crossing；
- sudden_start；
- occluded_emergence；
- intent_ambiguous；
- semantic_distractor。

## 统计协议

- 固定测试集不参与训练和调参；
- 每个方法使用相同测试 seed；
- 报告 mean 和 95% confidence interval；
- 对关键指标使用 paired bootstrap 或 paired test；
- 保留失败案例可视化。
