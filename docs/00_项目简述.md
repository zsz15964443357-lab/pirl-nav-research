# 00 项目简述

## 项目一句话

PIRL-Nav 研究的是：在潜在运动意图不确定环境中，让无人机通过预测式意图风险约束强化学习，在风险真正发生前提前、平滑、低绕行地避险。

## 核心问题

普通动态避障通常处理已经运动的障碍物，例如当前速度、距离、TTC 已经显示危险的对象。但低空无人机场景中还有一类更隐蔽的风险：

- 当前静止但可能突然启动的行人、车辆、叉车、推车、移动机器人；
- 门口、拐角、货架后可能突然出现的对象；
- 当前轨迹相似但未来可能 stay、cross、start、yield 的多意图对象。

本项目将这类问题定义为：**latent motion intent uncertainty 下的无人机导航**。

## 核心假设

如果策略能够接收并优化动作相关的未来意图风险，那么它会比 TTC reward、语义风险 reward、普通 PPO、普通 safety shield 更早开始规避风险，并且减少 near miss、risk exposure 和 shield dependence。

## 推荐贡献点

1. 提出潜在运动意图不确定环境下的 UAV navigation 问题定义。
2. 提出 action-conditioned predictive intent-risk cost。
3. 提出 intent-risk constrained RL 框架。
4. 使用 shield intervention 作为训练信号，降低策略对末端安全修正的依赖。
5. 构建可视化审查的 benchmark 和行为级指标体系。

## 非目标

- 不做端到端 RGB 强化学习。
- 不把语义分割作为论文主创新。
- 不把 PyBullet / Gazebo / ROS2 作为论文主创新。
- 不声称第一版已经具备真实无人机飞行能力。

## 成功标准

只有当 PIRL-Nav 在以下指标上相对 baseline 有一致趋势时，项目才具备论文价值：

- near miss 降低；
- integrated risk exposure 降低；
- Anticipation Time 提前；
- Shield Dependence Index 降低；
- success rate 不明显下降；
- detour 和 jerk 不显著恶化。
