## 文档优先原则

所有实现必须能追溯到：

- docs 中的研究决策；
- experiments/scenario_specs 中的场景规格；
- codex_tasks 中的当前阶段任务；
- review_checklists 中的审查规则。

## 版本管理原则

不要提交：

- 大规模训练日志；
- wandb / tensorboard 运行目录；
- rosbag；
- 大型视频；
- 临时图片；
- build 目录。

需要保留的代表性图像后续可放到：

```text
docs/figures/
```

但必须经过人工筛选。
