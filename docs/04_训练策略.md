# 04 训练策略

## 总体顺序

训练不要一开始就把 GRU predictor、PPO-Lagrangian、risk field、shield internalization、ROS2 全部打开。必须阶段化推进。

```text
场景与指标单元测试
  ↓
普通 PPO baseline
  ↓
TTC / semantic risk baseline
  ↓
PPO-Lagrangian without intent
  ↓
rule-prior PIRL-Nav
  ↓
GRU intent predictor
  ↓
PIRL-Nav full
  ↓
Gazebo / ROS2 验证
```

## 课程学习

### Curriculum 1：基础导航

- 空场景或少量静态障碍；
- 无感知噪声；
- 宽通道；
- 目标：学会到达目标且不碰撞。

### Curriculum 2：普通动态避障

- 加入 dynamic_crossing；
- 对象速度较低；
- 无 sudden-start；
- 目标：建立普通动态避障能力。

### Curriculum 3：潜在启动风险

- 加入一个 sudden_start 对象；
- 启动时间较晚；
- 语义准确；
- 目标：学会在对象启动前产生适度响应。

### Curriculum 4：遮挡与多意图

- 加入 occluded_emergence 和 intent_ambiguous；
- 多对象；
- 中等噪声；
- 目标：学习多模态风险。

### Curriculum 5：鲁棒性训练

- 语义错分；
- tracking noise；
- dropout；
- delay；
- control disturbance。

## Reward

基础 reward 保持简单：

```text
R = progress_reward
  + goal_reward
  - collision_penalty
  - time_penalty
  - jerk_penalty
  - detour_penalty
```

## Cost

第一版主约束只保留两个：

```text
C_near
C_risk
```

稳定后再加入：

```text
C_shield
```

不建议一开始同时引入太多 Lagrangian cost，否则训练曲线难解释。

## Shield 使用策略

训练早期：强 shield，防止大量碰撞。

训练中期：记录 policy action 与 safe action，加入 shield intervention cost。

训练后期：降低 shield 依赖，分别评估 policy-only 和 policy+shield。

## 基线方法

至少包括：

1. Vanilla PPO；
2. PPO + TTC reward；
3. PPO + semantic risk reward；
4. PPO-Lagrangian without intent；
5. PIRL-Nav rule prior；
6. PIRL-Nav full。
